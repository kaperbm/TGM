M 6a.0.1: Generische Utility-Methoden

Completion requirements
To do: Make a submission Done: Receive a grade
Due: Friday, 17 February 2023, 5:00 PM

Schreibe eine Klasse GenericUtils mit einer Klassenmethode namens umdrehen, die ein Array eines generischen Typs als Parameter übernimmt. Die Methode soll dieses Array umdrehen (so dass das erste Element an letzter Stelle steht und das letzte Element an erster Stelle steht). Das Umdrehen soll in-place passieren, es soll also kein neues Array erzeugt werden. (generische Arrays erzeugen geht nämlich, ähnlich wie Konstruktoren eines generischen Typs aufrufen, nicht (so leicht).)
